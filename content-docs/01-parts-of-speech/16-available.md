# 🔄 "Available" 的用法

> Synced from web component: `src/components/content/MultiPos/AvailableContent.tsx`

## Web Lesson Text

### 💡 为什么学这个？

### 📝 主要用法

### 1. 形容事物 (Describing Things)

### 2. 形容人 (Describing People)

"Available" 是一个非常实用的形容词，可以用来描述物品（可获得的）和人（有空的）。学会它能让你的日常沟通更顺畅，无论是购物、预约还是安排会议。

意思是“可用的”或“可获得的”。用来表示某物可以被购买、找到或使用。

意思是“有空的”或“有时间的”。用来表示某人没有被占用，可以做某事。

## Examples

1. `Are there any tickets **available**?`
   - 还有票吗？
2. `This dress is not **available** in your size.`
   - 这件连衣裙没有你的尺码。
3. `The doctor is not **available** right now.`
   - 医生现在没空。
4. `Are you **available** for a call tomorrow?`
   - 你明天有空打电话吗？

## Selector / Fill Practice

1. `I'm sorry, the doctor is not ____ right now.`
   - Choices: **available** / open / present
   - Chinese: 对不起，医生现在没空。
2. `Are there any seats ____ on this flight?`
   - Choices: **available** / empty / free
   - Chinese: 这次航班还有空位吗？
3. `The new software update is now ____ for download.`
   - Choices: **available** / ready / public
   - Chinese: 新的软件更新现在可以下载了。
4. `I'm afraid I'm not ____ for a meeting on Friday.`
   - Choices: **available** / free / possible
   - Chinese: 恐怕我周五没时间开会。
5. `Is this shirt ____ in blue?`
   - Choices: **available** / existing / made
   - Chinese: 这件衬衫有蓝色的吗？
6. `The information is made publicly ____ on their website.`
   - Choices: **available** / shown / visible
   - Chinese: 这些信息在他们的网站上是公开的。
7. `The manager is ____ to talk if you have any questions.`
   - Choices: **available** / ready / willing
   - Chinese: 如果你有任何问题，经理有空可以谈谈。
8. `There are no tickets ____ for Saturday's show.`
   - Choices: **available** / left / remaining
   - Chinese: 周六演出的票已经没有了。
