# Value

## 一句话先记住

`value` 有时是名词，有时是动词。

## 两种用法

### 1. 名词

- The value of time is great.
- This ring has high value.

这里 `value` 是“价值”，所以是名词。

### 2. 动词

- I value your help.
- We value honesty.

这里 `value` 是“重视”，所以是动词。

## 快速练习 1

判断 `value` 是名词还是动词：

1. We value good teachers.
2. The value is clear.
3. I value your opinion.
4. Money has value.

## 快速练习 2

下面句子里的 `value` 是什么？

1. The value of health is huge.
   - A. 名词
   - B. 动词

2. They value teamwork.
   - A. 名词
   - B. 动词

## 答案

### 快速练习 1

1. 动词
2. 名词
3. 动词
4. 名词

### 快速练习 2

1. A
2. B
