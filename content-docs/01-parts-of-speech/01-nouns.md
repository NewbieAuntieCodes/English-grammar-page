# 📚 Nouns (名词)

> Synced from web component: `src/components/content/PartsOfSpeech/NounsContent.tsx`

## Web Lesson Text

### 💡 为什么学名词？

### 📝 Examples

名词是句子的主角！它们告诉我们句子在谈论“谁”或“什么”。掌握名词是构建任何句子的第一步。

## Examples

1. `The **dog** barks.`
   - 这只狗在叫。（dog 是一个‘事物’）
2. `**Maria** is a **student**.`
   - 玛利亚是一名学生。（Maria 和 student 都是‘人’）
