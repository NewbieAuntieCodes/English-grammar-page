# 主谓宾宾补 / Subject + Verb + Object + Complement

## 一句话先记住

`主谓宾宾补 = 谁 + 做什么 + 对谁 / 什么 + 这个宾语后来是什么 / 怎么样。`

## 怎么认这个句型？

先找出主谓宾。

如果宾语后面还有一个部分，继续说明宾语，这就是宾补。

也可以先问：

- 这个词是在说明主语吗？
- 还是在说明宾语？

如果是在说明宾语，常常就是宾补。

## 例句 1：宾补是名词

1. We call him Tom.
2. They named the baby Lucy.
3. The class chose Emma monitor.

可以这样看：

- We / call / him / Tom
- They / named / the baby / Lucy
- The class / chose / Emma / monitor

这里最后的词都在说明宾语是谁、叫什么、是什么身份。

## 例句 2：宾补是形容词

1. The joke made him happy.
2. She painted the wall white.
3. We found the book useful.

可以这样看：

- The joke / made / him / happy
- She / painted / the wall / white
- We / found / the book / useful

这里最后的词都在说明宾语怎么样。

## 最短提醒

- 宾补是在说明宾语，不是在说明主语
- 宾补不是第二个宾语
- 宾补常是名词或形容词

## 一个容易混的对比

`She is happy.`

- `happy` 在说明 `She`
- 这是主系表

`The news made her happy.`

- `happy` 在说明 `her`
- 这是主谓宾宾补

## 快速练习 1

找出下面句子里的宾语和宾补：

1. We call our dog Max.
2. The teacher made us quiet.
3. They elected him president.
4. I found the test easy.

## 快速练习 2

判断下面句子是不是主谓宾宾补：

1. She likes music.
2. We named the cat Mimi.
3. The movie made me sad.
4. He is tall.

## 答案

### 快速练习 1

1. 宾语：our dog；宾补：Max
2. 宾语：us；宾补：quiet
3. 宾语：him；宾补：president
4. 宾语：the test；宾补：easy

### 快速练习 2

1. 不是
2. 是
3. 是
4. 不是

## 小结

看见宾语后面还有一个词时，不要急着当第二个宾语。

先看它是不是在补充说明宾语。
