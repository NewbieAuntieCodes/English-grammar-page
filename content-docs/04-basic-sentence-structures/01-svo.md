1. `I____apples.`
   - Choices: **like** / am / happy
   - Chinese: 我喜欢苹果。

   - Choices: **reads** / is / reading
   - Chinese: 她读书。
3. `____play games.`
   - Choices: **We** / Is / Are
   - Chinese: 我们玩游戏。

   - Choices: **music** / beautiful / run
   - Chinese: 他热爱音乐。
