# 基础句型结构 / Basic Sentence Structures

## 这一组在学什么？

这一组在学：

`一个完整句子最常见的骨架是什么。`

前面你已经学过：

- 词性
- 句子成分

现在要把这些东西放进真正的句子里。

## 先记住这 3 个核心句型

1. `主谓宾`：I like English.
2. `主系表`：She is happy.
3. `主谓宾宾补`：We call him Tom.

## 先记住的最短区别

- `主谓宾`：动作后面有对象
- `主系表`：后面是在说明主语
- `主谓宾宾补`：后面是在补充说明宾语

## 怎么学这一组？

看句子时，先问这 3 个问题：

1. 主语是谁 / 什么？
2. 中间是动作，还是系动词？
3. 最后那个部分是在说明宾语，还是说明主语？

## 先看一个总例子

### 例句 1

`I like music.`

- `I` 主语
- `like` 谓语
- `music` 宾语

这是 `主谓宾`。

### 例句 2

`The soup tastes good.`

- `The soup` 主语
- `tastes` 系动词
- `good` 表语

这是 `主系表`。

### 例句 3

`They named the baby Lucy.`

- `They` 主语
- `named` 谓语
- `the baby` 宾语
- `Lucy` 宾补

这是 `主谓宾宾补`。

## 这一组最容易错的地方

- 不要看到动词就一律当 `主谓宾`
- 不要把表语和宾语混在一起
- 不要把宾补看成第二个宾语
- 句子扩展时，不要把信息乱放位置

## 学完这一组后，你应该会什么？

- 能判断一个简单句属于哪种常见句型
- 能分清宾语、表语、宾补
- 能把短句慢慢扩成更完整的句子

## 模块小练习

判断下面句子属于哪种句型：

1. She likes coffee.
2. My father is busy.
3. We call the dog Max.
4. The flowers smell sweet.

## 答案

1. 主谓宾
2. 主系表
3. 主谓宾宾补
4. 主系表
