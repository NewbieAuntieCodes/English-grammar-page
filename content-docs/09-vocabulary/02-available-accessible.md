# 📖 Available vs. Accessible

> Synced from web component: `src/components/content/Vocabulary/AvailableAccessibleContent.tsx`

## Web Lesson Text

### 💡 为什么学这个？

### 📝 主要用法

### 1. Available (可获得的 / 有空的)

### 2. Accessible (可进入的 / 可理解的)

"Available" 和 "Accessible" 意思相近，都表示“可以得到”或“可以使用”，但它们的侧重点不同，很容易混淆。

**侧重点:** 某物是否可以被【购买、使用或找到】；某人是否【有时间】。

**侧重点:** 是否容易【进入、到达或理解】，常常暗含“无障碍”的意思。

## Examples

1. `The book is **available** in the library.`
   - 这本书在图书馆可以借到。
2. `The manager is not **available** until 2 PM.`
   - 经理下午两点才有空。
3. `The new building is **accessible** to wheelchair users.`
   - 新大楼对轮椅使用者是无障碍的。
4. `The website is **accessible** from any device.`
   - 这个网站可以从任何设备访问。

## Selector / Fill Practice

1. `I'm sorry, the manager is not ____ right now.`
   - Choices: **available** / accessible
   - Chinese: 对不起，经理现在没空。
2. `The new ramp makes the library ____ to everyone.`
   - Choices: **accessible** / available
   - Chinese: 新的坡道使图书馆对所有人开放。
3. `Are there any tickets ____ for tonight's show?`
   - Choices: **available** / accessible
   - Chinese: 今晚的演出还有票吗？
4. `The website should be ____ on both mobile and desktop.`
   - Choices: **accessible** / available
   - Chinese: 网站应该在手机和桌面上都可以访问。
5. `Is this shirt ____ in a larger size?`
   - Choices: **available** / accessible
   - Chinese: 这件衬衫有更大号的吗？
6. `The remote village is not easily ____ by car.`
   - Choices: **accessible** / available
   - Chinese: 这个偏远的村庄开车不易到达。
7. `Dr. Lee will be ____ to see you at 3 PM.`
   - Choices: **available** / accessible
   - Chinese: 李医生下午3点有空见你。
8. `We need to make our content more ____ to people with disabilities.`
   - Choices: **accessible** / available
   - Chinese: 我们需要让我们的内容更容易被残疾人士获取。
