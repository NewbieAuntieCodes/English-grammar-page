# ⚖️ 'as' 相关短语用法

## Web Lesson Text

### 💡 为什么学这个？

### 📝 主要用法

### 1. 肯定句：as + 形容词/副词 + as

### 2. 否定句：not as/so + 形容词/副词 + as

### 3. as long as (只要)

### 4. as well as (也, 和)

包含 "as" 的短语是英语中最基本的比较和条件句型之一。本节课我们学习 "as...as" (和...一样), "as long as" (只要) 和 "as well as" (也, 和)。掌握它们能让你的表达更丰富、更精确。

在否定句中，第一个 as 可以用 so 替换，意思不变。

用来引导一个条件状语从句，表示“只要...就...”。

用来连接两个并列的成分，类似 "and"，但更强调前者。当 `A as well as B` 作主语时，谓语动词的单复数由 A 决定。

## Examples

1. `She is **as tall as** her brother.`
   - 她和她哥哥一样高。(形容词)
2. `He is **not as tall as** his father.`
   - 他不如他父亲高。
3. `You can use my car **as long as** you drive carefully.`
   - 只要你小心驾驶，你就可以用我的车。
4. `She is smart **as well as** beautiful.`
   - 她既聪明又漂亮。
5. `The teacher, **as well as** the students, **is** excited.`
   - 老师和学生们都很兴奋。(谓语动词 is 跟随主语 the teacher)

## Selector / Fill Practice

1. `He is ____ his brother.`
   - Choices: **as tall as** / taller / as tall
   - Chinese: 他和他的哥哥一样高。
2. `This book is ____ that one.`
   - Choices: **not as interesting as** / more interesting than / not interesting
   - Chinese: 这本书不如那本有趣。
3. `She sings ____ a bird.`
   - Choices: **as beautifully as** / more beautiful than / beautiful as
   - Chinese: 她唱歌像鸟儿一样动听。
4. `I can run ____ you.`
   - Choices: **as fast as** / faster than / as fast
   - Chinese: 我能跑得和你一样快。
5. `He is not ____ his father.`
   - Choices: **so strong as** / stronger than / as strong
   - Chinese: 他不如他父亲强壮。
6. `My car is ____ yours.`
   - Choices: **as expensive as** / more expensive than / so expensive
   - Chinese: 我的车和你的一样贵。
7. `She doesn't speak English ____ her sister.`
   - Choices: **as fluently as** / more fluently than / as fluent
   - Chinese: 她说英语不如她姐姐流利。
8. `The weather today is ____ yesterday.`
   - Choices: **as good as** / better than / as well as
   - Chinese: 今天的天气和昨天一样好。
9. `You can watch TV ____ you finish your homework.`
   - Choices: **as long as** / as well as / as soon as
   - Chinese: 只要你完成作业，你就可以看电视。
10. `She is smart ____ beautiful.`
   - Choices: **as well as** / as long as / as good as
   - Chinese: 她既聪明又漂亮。
11. `We will go to the park ____ it doesn't rain.`
   - Choices: **as long as** / as well as / as much as
   - Chinese: 只要不下雨，我们就会去公园。
12. `The captain, ____ the players, was happy.`
   - Choices: **as well as** / as long as / and
   - Chinese: 队长和队员们都很高兴。
