## Web Lesson Text

### 💡 为什么学这个？

### 📝 主要结构

### 1. What 感叹句 (强调名词)

### 2. How 感叹句 (强调形容词/副词)

感叹句是用来表达强烈情感（如惊讶、赞美、喜悦）的句子。学会使用 "What" 和 "How" 开头的感叹句，能让你的英语表达更富有感情色彩，听起来更生动！

结构：**What + a/an + (形容词) + 单数可数名词!** 或 **What + (形容词) + 复数/不可数名词!**
这个结构的核心是强调“一个多么...的【名词】啊！”

结构：**How + 形容词/副词 + (主语 + 谓语)!**
这个结构的核心是强调“多么地...啊！”

## Examples

1. `**What a beautiful day!**`
   - 多美好的一天啊！ (强调的是 a day)
2. `**What beautiful flowers!**`
   - 多美的花儿啊！(强调的是 flowers, 复数名词前不用 a/an)
3. `**How beautiful** the flower is!`
   - 这朵花多美啊！(强调的是 beautiful)
4. `**How fast** he runs!`
   - 他跑得多快啊！(强调的是 fast)

## Sentence Builder Practice

1. Words: `What a` / `beautiful` / `garden!`
   - Correct: `What a beautiful garden!`
   - Chinese: 多么美丽的花园啊！
2. Words: `How fast` / `he` / `runs!`
   - Correct: `How fast he runs!`
   - Chinese: 他跑得真快啊！
3. Words: `What an` / `interesting book` / `this is!`
   - Correct: `What an interesting book this is!`
   - Chinese: 这是一本多么有趣的书啊！
4. Words: `How cold` / `it is` / `today!`
   - Correct: `How cold it is today!`
   - Chinese: 今天天气真冷啊！
5. Words: `What a` / `delicious` / `meal!`
   - Correct: `What a delicious meal!`
   - Chinese: 多么美味的一餐啊！
6. Words: `How well` / `she` / `sings!`
   - Correct: `How well she sings!`
   - Chinese: 她唱得多好啊！
7. Words: `What a surprise` / `to see` / `you here!`
   - Correct: `What a surprise to see you here!`
   - Chinese: 在这里见到你真是个惊喜！
8. Words: `How quickly` / `time` / `flies!`
   - Correct: `How quickly time flies!`
   - Chinese: 时间过得真快啊！
