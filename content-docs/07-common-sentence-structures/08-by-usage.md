# 🤝 介词 'by' 的用法

> Synced from web component: `src/components/content/CommonStructures/ByUsageContent.tsx`

## Web Lesson Text

### 💡 为什么学这个？

### 📝 主要用法

### 1. 表示方式、方法 (Method/Means)

### 2. 引出被动语态的动作执行者 (Agent in Passive)

### 3. 表示截止时间 (Deadline)

### 4. 表示位置“在...旁边” (Proximity)

"by" 是一个功能强大的介词。它最常见的用法是引出被动语态中的动作执行者，还可以表示方式、途径、在...旁边或截止时间。掌握它对于理解和使用更复杂的句子至关重要！

## Examples

1. `I go to work **by subway**.`
   - 我乘地铁上班。(交通方式)
2. `You can pay **by cash**.`
   - 你可以用现金支付。(支付方式)
3. `The novel was written **by him**.`
   - 这本小说是他写的。
4. `Please finish it **by Monday**.`
   - 请在周一前完成它。
5. `He lives in a house **by the sea**.`
   - 他住在一所海边的房子里。

   - Chinese: 这个蛋糕是我妈妈做的。

   - Correct: `I go to school by bus`

   - Correct: `You can pay by credit card`
   - Chinese: 你可以用信用卡支付。
4. Words: `Please finish` / `the report` / `by Friday`
   - Correct: `Please finish the report by Friday`
   - Chinese: 请在周五前完成报告。
5. Words: `The house` / `is` / `by the river`
   - Correct: `The house is by the river`
   - Chinese: 房子在河边。
6. Words: `He learned English` / `by watching movies`
   - Correct: `He learned English by watching movies`
   - Chinese: 他通过看电影学会了英语。
7. Words: `The window` / `was broken` / `by the children`

   - Chinese: 窗户被孩子们打破了。
8. Words: `You must arrive` / `by 10 AM`
