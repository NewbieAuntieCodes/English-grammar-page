# 🏃 动词-ing的用法 (Usage of V-ing)

> Synced from web component: `src/components/content/CommonStructures/GerundsContent.tsx`

## Web Lesson Text

### 💡 什么是动词-ing？

### 1. 作主语 (As Subject)

### 2. 作宾语 (As Object)

### 3. 作介词宾语 (As Object of a Preposition)

### 4. 作表语 (As Complement)

### 5. 作状语 (As Adverbial) - 表示伴随

### 6. 作定语 (As Attributive) - 后置修饰

### 结构公式

动词的-ing形式是一个“多面手”！当它像【名词】一样使用时，我们称之为动名词 (Gerund)；当它像【形容词】或【副词】一样使用时，我们称之为现在分词 (Present Participle)。它可以在句子中扮演多种角色，学会它能让你的表达更流畅、更地道！

把一个“动作”当作句子的“主角”。

放在 `enjoy`, `finish`, `mind`, `practice`, `avoid`, `suggest` 等动词后面，作为动作的对象。

放在 `at`, `for`, `in`, `of`, `without` 等介词后面。

放在系动词 (如 is, am, are) 后面，用来解释说明主语是什么。

表示伴随情况，说明主语在做主要动作的同时，还在做什么。它通常用逗号与主句隔开。

放在名词后面，作后置定语，用来修饰该名词，功能类似于一个定语从句。

## Examples

1. `**Reading** is fun.`
   - 阅读很有趣。
2. `I enjoy **reading**.`
   - 我享受阅读。
3. `She is good at **drawing**.`
   - 她擅长画画。
4. `My hobby is **collecting** stamps.`
   - 我的爱好是集邮。
5. `He sat on the sofa, **watching TV**.`
   - 他坐在沙发上，看着电视。
6. `The man **talking to John** is my teacher.`
   - 正在和约翰说话的那个男人是我的老师。(= The man who is talking to John...)
