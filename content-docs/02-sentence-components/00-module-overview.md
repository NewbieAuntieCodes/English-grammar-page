# 句子成分 / Sentence Components

## 这一组在学什么？

词性是在看“这个词是什么”。

句子成分是在看“这个词或短语在句子里做什么”。

这一组先学 6 个最常见的句子成分：

1. 主语 Subject
2. 谓语 Predicate
3. 宾语 Object
4. 定语 Attributive
5. 状语 Adverbial
6. 补语 Complement

## 先记住这一句

`句子成分看工作，不看身份。`

比如：

- `boy` 是名词，这是词性
- 在句子里它可以做主语，也可以做宾语，这是句子成分

## 怎么学这一组？

每一节都按这个顺序看：

1. 先记一句最短定义
2. 看 2 到 3 个例句
3. 先自己找，再看答案
4. 不要求一次全会，先会找最明显的

## 先看一个总例子

句子：

`The clever boy opened the door quietly.`

可以这样看：

- `The clever boy` 是主语
- `opened` 是谓语
- `the door` 是宾语
- `clever` 是定语
- `quietly` 是状语

## 这一组最容易错的地方

- 不要把“词性”和“句子成分”混在一起
- 不要以为一个成分一定只有一个单词
- 不要看到动词就只会说“动词”，还要问它是不是谓语
- 不要看到形容词就只会说“形容词”，还要看它是不是定语或补语

## 学完这一组后，你应该能做到

- 在简单句里找出主语、谓语、宾语
- 在短句里认出定语和状语
- 知道补语是在“补充说明”
- 能把句子拆成几个主要部分，而不是只看单个词
