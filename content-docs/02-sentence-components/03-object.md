## Web Lesson Text

### 💡 什么是宾语？

### 📝 Examples

宾语是动作影响到的人或事。它常跟在实义动词后面，回答“做了什么”或“对谁做”。

## Examples

1. `She reads **a book**.`
   - 她在读书。（她读了什么？一本书。）
2. `He loves **her**.`
   - 他爱她。（他爱谁？她。）
